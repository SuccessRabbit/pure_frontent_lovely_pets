# Excel 配置文件说明

## 文件结构

```
excel/
├── cards.xlsx          # 卡牌配置表（主要编辑文件）
├── synergies.xlsx      # 羁绊配置表（可选）
└── README.md           # 本说明文件
```

## cards.xlsx 表格格式

### 必填列

| 列名 | 类型 | 说明 | 示例 |
|------|------|------|------|
| ID | 文本 | 唯一标识符 | pet_001, worker_001 |
| 名称 | 文本 | 卡牌显示名称 | 新晋柯基 |
| 类型 | 枚举 | 见下方类型列表 | 实体-萌宠 |
| 费用 | 数字 | 小罐头消耗 | 2 |
| 描述 | 文本 | 卡牌描述文本 | 可爱但容易焦虑的新人 |
| 稀有度 | 枚举 | 普通/稀有/史诗/传说 | 普通 |

### 可选列（根据卡牌类型）

| 列名 | 适用类型 | 说明 | 示例 |
|------|----------|------|------|
| 收益 | 实体卡 | 每回合产出小红心 | 2 |
| 压力 | 实体-萌宠 | 每回合累积压力 | 1 |
| 压力上限 | 实体-萌宠 | 触发拆家阈值 | 3 |
| 特殊效果 | 所有 | 效果描述 | 相邻牛马效率+10% |
| 标签 | 所有 | 用于羁绊匹配，逗号分隔 | cat,cute |

## 卡牌类型列表

### 实体卡（部署后留在场上）
- **实体-萌宠**：主要产出单位，会累积压力
- **实体-牛马**：辅助产出单位，可献祭
- **实体-设施**：工位、设备等

### 指令卡（使用后进入弃牌堆）
- **指令-增益**：提升收益、效率
- **指令-减压**：降低压力、安抚萌宠
- **指令-功能**：特殊功能（交换位置、抽牌等）

### 状态卡（负面卡牌）
- **状态-负面**：黑粉弹幕、怨气等

## 特殊效果语法

支持以下格式的自动解析：

```
相邻牛马效率+10%
全场压力-1
压力>2时收益翻倍
拆家范围扩大到3x3
可献祭
```

复杂效果可以用自然语言描述，程序会保留原始文本。

## 使用流程

1. **编辑 Excel**
   - 打开 `cards.xlsx`
   - 按照上述格式填写卡牌数据
   - 保存文件

2. **运行转换脚本**
   ```bash
   npm run convert:cards
   ```

3. **检查输出**
   - 配置文件：`config/cards.json`
   - 类型定义：`src/types/cards.generated.ts`

4. **验证结果**
   - 检查控制台输出的统计信息
   - 确认没有错误提示

## 注意事项

⚠️ **重要提示：**

1. **ID 必须唯一**：不能有重复的 ID
2. **类型必须精确匹配**：严格按照列表中的文本填写
3. **数值必须合理**：费用、收益等不能为负数
4. **保存为 .xlsx 格式**：不要使用 .xls 或 .csv
5. **不要修改表头**：第一行的列名不能改动

## 数值平衡参考

### 费用与收益比例
- **1 费卡**：收益 1-2，适合前期铺场
- **2 费卡**：收益 2-3，中期主力
- **3 费卡**：收益 4-5，后期核心

### 压力设计
- **低压力（0-1）**：稳定型，适合长期运营
- **中压力（1-2）**：平衡型，需要定期维护
- **高压力（2+）**：高风险高收益，需要重点关注

### 稀有度分布建议
- **普通**：60-70%（基础卡池）
- **稀有**：20-30%（核心卡牌）
- **史诗**：5-10%（强力卡牌）
- **传说**：1-5%（游戏改变者）

## 常见问题

**Q: 转换脚本报错怎么办？**
A: 检查错误提示的行号，确认该行数据格式正确，特别注意类型和稀有度的拼写。

**Q: 可以添加自定义列吗？**
A: 可以，但转换脚本不会处理未定义的列。如需使用，需要修改脚本。

**Q: 如何批量修改卡牌数值？**
A: 直接在 Excel 中使用公式或批量替换功能，然后重新运行转换脚本。

**Q: 图片资源如何对应？**
A: 图片路径根据 ID 自动生成，例如 `pet_001` 对应 `/assets/cards/pets/pet_001.png`
