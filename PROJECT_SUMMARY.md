## ✅ 核心改进

### 🎨 新的卡牌系统

**卡牌不再是整张SVG，而是代码动态组装！**

```
卡牌组成 = 代码框架 + 角色插画 + 动态数据
```

#### 卡牌元素（自动生成）
1. **费用标识** - 右上角圆形，显示小罐头消耗
2. **稀有度标签** - 左上角，根据稀有度显示不同颜色
3. **角色插画** - 4:3比例，PNG→SVG自动fallback
4. **卡牌名称** - 居中显示
5. **属性图标** - 收益⭐、压力💢（实体卡专属）
6. **效果描述** - 从配置文件读取
7. **标签** - 显示前3个标签
8. **悬停效果** - 缩放、光效、阴影

#### 优势
- ✅ 完全可控的布局和样式
- ✅ 易于调整和扩展
- ✅ 支持动画和交互
- ✅ 设计师只需提供角色插画

### 与旧版本的区别

**旧版本（已废弃）：**
```
/assets/cards/pets/pet_001.svg  ← 整张卡牌
```
- 包含所有元素
- 难以修改
- 无法动态调整

**新版本（当前）：**
```
/assets/illustrations/pets/pet_001.svg  ← 只有角色
```
- 只包含角色插画
- 卡牌由代码组装
- 灵活可配置

## ✨ 使用示例

### 基础用法
```tsx
import { Card } from '@/components/Card';
import petCard from '@config/pets.json';

<Card
  card={petCard[0]}
  onClick={() => console.log('clicked')}
/>
```

### 禁用状态
```tsx
<Card
  card={card}
  disabled={true}
/>
```

### 自定义样式
```tsx
<Card
  card={card}
  className="transform scale-150"
/>
```
