# 运营操作指南

## 工作目标

运营人员主要负责把已经确定的卡牌内容录入后台，包括名称、描述、费用、稀有度、数值、资源、技能绑定和起始牌堆。除非策划明确要求，不建议直接改模板工作台里的 effectKind 或 operations。

## 新建卡牌

1. 进入 `卡牌与技能`。
2. 如果想沿用某类卡牌，先用类型筛选定位到目标类型。
3. 点击左侧 `新建`。
4. 在中间编辑区填写名称、费用、稀有度、标签和描述。
5. 如果是萌宠或员工卡，继续填写收益、初始压力和压力上限。
6. 选择卡面图片资源和插画资源。
7. 如果是场上实体卡，选择 3D 模型配置，并确认预览正常。

注意：新建卡牌会使用当前筛选类型作为默认类型；如果筛选是“全部类型”，默认创建 `action_utility`。

## 复制已有卡牌

1. 在左侧选中一张相似卡牌。
2. 点击 `复制`。
3. 系统会复制卡牌字段和已有技能绑定。
4. 修改名称、描述、数值和资源，避免保留旧卡含义。

复制适合做同系列卡牌，尤其是技能结构相同但参数不同的卡。

## 编辑卡牌数值

`卡牌与技能` 有两种布局：

- 原有布局：适合逐张卡做完整编辑。
- 列表布局：适合横向对比费用、稀有度、收益、压力、压力上限和可弃置状态。

列表布局里，简单数值可以直接在表格中修改；描述、资源和技能绑定需要点击 `深度编辑`。

## 绑定技能

1. 选中卡牌。
2. 在 `Skills` 区域点击 `添加技能绑定`。
3. 选择技能模板。
4. 设置排序、启用状态和参数。
5. 查看 `玩家可见摘要` 是否符合预期。
6. 查看右侧 `Skill Flow`，确认触发、目标和效果方向正确。

如果模板下拉中没有目标模板，通常说明该模板的 `scope` 不支持当前卡牌类型，需要策划在模板工作台调整。

## 修改技能参数

技能参数来自模板定义。运营只需要填写表单暴露的参数，例如倍率、数量、目标卡牌或作用单位。后台会根据模板生成玩家可见摘要。

如果出现“必填参数未完成”或保存时提示阻断问题，需要补齐参数或联系策划检查模板。

## 配置起始牌堆

1. 进入 `全局参数`。
2. 在 `起始牌堆配置` 中选择卡牌。
3. 点击 `添加这张卡`。
4. 用 `+`、`-` 或数量输入框调整张数。
5. 查看右侧牌堆概览，确认总张数和按类型统计。

如果起始牌堆为空，运行时会回退为默认“全卡各 2 张”。

## 保存与验证

1. 修改完成后点击左上角 `保存`。
2. 如果顶部提示模板阻断问题，后台会自动跳到模板工作台，需要先修复。
3. 保存成功后，CSV 会被回写并重新编译。
4. 回到游戏页验证卡牌显示、技能摘要、资源和起始牌堆。

## 常见问题

- 按钮不能编辑：当前不是本地可写环境，或 Admin API 没有启动。
- 资源选不到：资源文件可能没有放到 `public/assets/` 对应目录，或卡牌类型筛选了不匹配的资源目录。
- 保存被阻断：通常是模板或技能绑定参数不完整，需要查看顶部提示和模板工作台校验结果。
- 游戏页没更新：确认保存成功，并刷新游戏页；本地热更新依赖 Admin API 事件。
