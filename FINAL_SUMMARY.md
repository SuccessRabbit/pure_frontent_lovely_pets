## 🎨 核心特色

### 1. 动态卡牌组装

**卡牌不是一张完整的图片，而是由多个元素组成：**

```tsx
<Card card={cardData} />
```

**自动生成：**
- 费用标识（右上角圆形）
- 稀有度标签（左上角）
- 角色插画（4:3比例）
- 卡牌名称
- 属性图标（收益⭐、压力💢）
- 效果描述
- 标签
- 悬停效果

### 2. 智能资源加载

```
PNG优先 → PNG不存在 → 自动SVG → 无缝显示
```

**优势：**
- 开发时无需等待图片
- 随时替换为PNG
- 零配置

### 3. 完整的设计文档

从技术选型到玩法设计，一应俱全。

## 💡 使用方式

### 卡牌组件
```tsx
import { Card } from '@/components/Card';
import petCard from '@config/pets.json';

<Card
  card={petCard[0]}
  onClick={() => console.log('clicked')}
  disabled={false}
/>
```

### 添加自定义插画
```bash
# 将PNG放入对应目录
public/assets/illustrations/pets/pet_001.png

# 系统自动优先加载PNG
```
